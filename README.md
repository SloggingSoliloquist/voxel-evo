# voxel-evo
Autonomous design of morphology and control parameters of voxel-based soft robots through the use of evolutionary algorithms. 

<img width="400" height="184" alt="robot" src="https://github.com/user-attachments/assets/c4171251-d5b9-4ed1-bb32-8822a11ca9a2" />

Employs the use of Compositional Pattern Producing Networks, Neuro Evolution of Augmenting Topologies(NEAT), Proximal Policy Optimization and Lamarckian Policy transfer.
Implemented using Pygame and Pymunk for the simulation engine and stablebaselines for PPO training.
